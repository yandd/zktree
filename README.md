zktree
======

Display znodes tree of zookeeper like command `tree`.

```
/
├── YINSHI.MONITOR.ALIVE.CHECK
├── tsdb
│   └── RO:20120101000000:20150203192000
│       └── 0:8192
├── driview
│   ├── master
│   │   └── m1
│   └── RW:10003:MASTER:192.168.1.14:7501:7502:20150512190000:-1
├── root
├── zookeeper
│   └── quota
└── a
```

Require install **zookeeper c libs**.
